## 🚀 Steps to run the app

### Prerequisites

- **Node >= 16**
- **Yarn**

### Clone & Run

```bash
git clone https://github.com/ahsanakhtar91/financing-request-creator.git
cd financing-request-creator
yarn install
yarn start
Access the app at http://localhost:3000
```
